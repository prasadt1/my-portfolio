## How to Capture

1. Use browser dev tools (Chrome/Firefox)
2. Set viewport to 1920x1080 for desktop, 375x667 for mobile
3. Enable dark mode for some screenshots
4. Use browser extension like "Full Page Screen Capture" for long pages
5. Save as PNG with high quality

## Notes

- All screenshots should be taken in production mode
- Include both light and dark mode variants where relevant
- Ensure no PII or sensitive data is visible
- Add annotations/arrows if needed to highlight key features
